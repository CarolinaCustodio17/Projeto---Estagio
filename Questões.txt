7 - O que consiste em um CRUD?

RESPOSTA 
CRUD consiste em quatro operações básicas para gerenciar elementos de dados persistentes, especialmente de dados relacionais e NoSQL :
C: Creade (criação)
R: Read (leitura)
U: Update (atualização) 
D: Delete (exclusão)
Cabe ressaltar que tanto o profissional de front-end quanto back-end  faz uso do CRUD, bem como o CRUD foi utilizado no teste prático do 11 ao 18.

19 - O que é Git? Para que serve?

RESPOSTA 
É um software  de controle de versão, portanto, rastreia e gerencia as alterações de um código além de ser um sistema distribuído,
possibilita trabalhar off-line pois permite repositório loal (com todas as versões do código/software), favorecendo o 
compartilhamento/sicronização/alterações feitas para o servidor remoto.
